## 前言



大三下学期对机器人的同学而言相比大三上又会轻松很多，我之前说大三上轻松，因为我没经历过大三下。当然这里指的是学业很轻松，几乎没课，以至于对假期都失去了感知，但是基本上到了这个时候无论是打算考研的同学，打算实习的同学，打算出国的同学，都在忙自己的事情；唯一轻松的就是决定校内保研而且稳保研线的同学——他们真的闲的没事情做。一个最大的感受就是，到了大三下，每个人好像都找到了自己的路，就这样分道扬镳了，我不禁总是回想到大家大一刚入学时的样子，感觉白驹过隙啊。到了这学期，正式意味着大学的时间比高中更长了。不过这里也不多做感慨了，毕竟是经验分享贴，写的比较早是因为刚刚考完人机这门课，实在忍不住想来吐槽一下；后面期末考的一两门课之后学完之后还会编辑更新的。打算后面升学顺利之后再写一个对本科三年的总结和思考吧。之后对暑研或者其他面试都会写一下面试经验。



和往常的三次分享一样，整理了这篇经验分享，希望能帮助到之后学习下列课程的同学们。在正式分享之前，同样几点我想先说明：



1. 我希望在类似之前经验分享的基础上，也尽量分享我学习的**平时学习和期末准备**的实际情况。每位同学都有自己独特的学习节奏和适应方式，我的经历最多只能帮大家粗略了解一门课可能需要投入的程度，权当是多一个参考视角，不足之处还请批评指正；



2. 部分课程若有必要，我会尽量分析我得分高/低时的具体原因。这些个人案例不一定有普适性，但或许能提供一些对照思考的素材；



3. 需要特别说明的是，这些内容主要反映的是我在**应试和得分**上的观察，会比较功利和急躁。如果同学们对课程内容有更深入的钻研需求，授课老师或其他学长学姐的经验可能会更加有用——这方面我还有很多需要学习的地方。



最后想说的是，毕竟学习方法因人而异，每个人都有自己的独到之处，我的这些经验仅供参考。如果这些分享能给你带来一点点启发，或是帮你节省一些摸索的时间，我就非常开心了。祝愿各位在新的一学期都能学习顺利，事事顺心！

### 人工智能与机器学习/3.5学分/吴争光、胡瑞芬



- **课程难度：** 4.0



- **分数：** 93/100

- **能否补天：** True



- **个人学习路线：** 平时：没有听课，之前有一定机器学习和大模型基础（不过感觉面对这个课还是颇为震撼）



大作业：实则等价于小作业，纯看想不想卷

- **个人感受：** 这个课程自动化是大三秋冬上，而且是长学期；我们是大三下上，春学期；



其实这门课没有大家说的那么那么烂，我觉得实际体验其实比上学期很多课，甚至比机器人实践、嵌入式、机械设计基础之类的都好很多，毕竟平常没有事情，期末实际需要投入的时间也不多，反而我觉得比需要整理A4的课要好。而且这门课对于机器人的同学来说，只是课本身烂，但是吴老师和胡老师人都很好。这门课之所以被喷主要是学的知识实在太烂了。



关于点名，吴老师没点过名，胡老师觉得来的人太少了还是点一下，形式是纸质签名就行，也不会对人头，而且中间去了的话老师也给签。胡老师说是签到会单独加分，不签到也不会扣分的。实验课不用去，占课时的。



但是这门课的ppt确实很难不让人吐槽一下：



- 2026年正处大模型发展演变的高峰，这门课程也适应时代潮流：虽然在知识点上与今天的 AI 不能说是紧密相连，也可以说是毫无关系，但 PPT 的制作确实是先进 AI 技术。

- 整套讲义形成了一种拼好课风格，集百家之长，摘自多个来源不明、风格各异的 PPT。

- 其章节安排也颇具特色，例如第二章结束后直接进入第六章，之后是第八章，然后是第四章，然后是第九章，接着仍然还是另一个第九章，唯一始终不变的是信息熵极低。

- 经典先不加说明地输出一大段内容，再附上一页“目录”来展示本次课要讲的知识点，但后续内容和重点分配不能说是一一对应吧，只能说是交相辉映。

- 此外，每一页的小标题也很少真正概括内容，善于频繁采用“XX（1）”“XX（2）”之类的机械堆叠方式，使得整体结构更加混乱。



使得很多知识点不知所云、左右脑互搏且绕来绕去，善于把简单的东西复杂化。且不说讲的东西老不老，就连基本的章节之间的脉络和关联都做不到；今年倒是有几章大模型了，但是直接ai做ppt是何意味？最终1200多页ppt有效信息量不足100页。所以平时上课没必要听，完全是浪费时间，且不说知识有用的有多少，现在有大量教程，有AI，自学的话会快很多。



先别管讲的是什么，我还是老样子捋一下思路**每一章讲了什么**，便于期末复习时候参照重点。事先说明，根据期末考试来看，冷门的小点也有几率被考到，所以我笔记上有的几乎都要背一背（具体的大家可以看历年卷风格），我这里只对最容易考的重点阐述。这里的章节名对应的是我的笔记的顺序，可能与上课顺序不同：



首先是**机器学习部分**：



**第一章**纯粹就是概论，只需要知道机器学习分为有监督、无监督、强化学习，此外还讲了懒惰学习和积极学习的区别；



**第二章**概念学习，最为重点的是Find-S算法的流程、候选消除算法的流程、enjoysport的实例数和假设数的计算、归纳偏置的概念；



**第三章**神经网络，感知器的输出、感知器法则和delta法则、梯度下降、反向传播（看起来不太可能考自己去手动算，都是概念）、过拟合和欠拟合的概念；



**第四章**决策树学习，决策树的表示和适用性，ID3算法流程、ID3的假设空间和归纳偏置；



**第五章**评估假设，其实就是数理统计学过的最后一章，感觉考的话也就是考个填空或选择，就一个置信区间的公式需要记住就行了，另外就是中心极限定理的概念考个选择；



**第六章**贝叶斯学习，主要是理解清楚贝叶斯学习目的是什么，捋清楚极大后验假设、极大似然假设的含义，会算条件概率的癌症那道题，何时用误差平方最小化、交叉熵最小化，最小长度描述准则的概念，贝叶斯最优分类器的原理、Gibbs算法看到就选2倍就行了，朴素贝叶斯分类器的流程，贝叶斯信念网的概念，EM算法的原理和Kmeans的原理，具体计算都不用管，其实这一章的脉络还是挺清晰的，每一个概念都是为了解决前一步的问题。



**第七章**MDP和强化学习，需要理解MDP和扫地机器人那道题，会自己写s,a,p,r，强化学习理解其如何训练和设计奖励，贝尔曼方程没见到考过但是其实很好理解；



**第八章**线性模型，这一章知道对数线性回归、岭回归与 Lasso 回归的那个L2和L1正则化，还有LDA的概念，以及基函数（这个倒很重要）的定义、优缺点等等；



接下来终于到了**人工智能部分**：



**第九章**搜索主要是BFS、UCS、DFS、DLS、Greedy、A\*、IDS、IDA\*的概念和搜索流程以及时间空间复杂度，启发式函数的设计原则和一致性、可容纳性和相关证明（如果没时间的话证明并无所谓），爬山法、模拟退火算法、遗传算法，都是知道概念和优缺点就行，重点是A\*算法的八数码问题要会写；



**第十章**纯AI，实时搜索，首先是哈希索引、位图索引、布隆过滤器、LSM树的概念，然后是RTA\*、D\*Lite、AD\*这三个概念和优缺点对比；知识驱动的看看就行，废话多；



**第十一章**智能体，就是很多概念纯背就行；任务规划里面任务分解比较重要；



**第十二章**博弈算法，最关键就是minimax和alpha-beta的流程，要会做会算，这个其实很简单；水平线效应和静止搜索；多智能体博弈知道概念就行，公式完全不用管；



**第十三章**大模型，五个特征必须要背，Transformer要理解并能自己阐述其结构和流程，Scaling law要会阐述并明白openai和Chinchilla的结论，公式不用管；分布式训练三种要理解和记忆；



**第十四章**架构基础，分词的几种算法要理解，传统NLP模型的概念和优缺点，Transformer概念和优点；微调的类别和更新哪些参数，优缺点；



**第十五章**是推理和训练区别，inference和reasoning要能分清，优化策略量化、剪枝、蒸馏等要会说；推理解码策略要清楚概念，推理性能延迟、吞吐量、成本三个基准，这里很多什么“经典论文”的表格，还有随便找的benchmark什么的都是AI生成别管，浪费时间；



**第十六章**多模态大模型，全是废话，知道什么是多模态，什么是MLLM的中文，以及preprocessor、encoder、projector、LLM分别是干什么的差不多自己能说就行。



总的来说，机器学习的部分更难理解，尤其是贝叶斯学习一章，需要多花些时间；人工智能部分除了alpha-beta算法基本是纯粹背诵，不过可能因为我是做大模型的所以对这里熟悉一点可能会有偏置（



今年有三次小测，吴老师测了两次，胡老师测了一次。胡老师小测之前还每周都提醒一次要小测，但是其实最后还是开卷的（可以看ppt但是不能用AI.jpg），平时不学感觉应该无所谓，小测人去了就行，实际上就是一个点名的作用。



**平时作业**就是mo平台的四个实验和一个实验作为大作业，今年胡老师还额外留了人工智能部分的所有ppt课后习题（没说必须手写，那就好办了，实际上没花多少时间，半小时弄完了）；主要还是这个mo平台的实验（不过听说这个实验尤其是实验报告也无人在意，主要还是看期末给分，所以不用特别卷）。据我观察每年的实验其实是不一样的，一共七个实验开放，前四个是个人实验，后面三个选一个作为大作业并汇报（所以实际上不卷的话和小作业实验是完全一样的量）；今年的个人实验是黑白棋、Kmeans、A股预测和垃圾分类，其中只有①和③需要写实验报告并且提供了模版：对于这四个实验都建议先了解基本原理再让AI写，不然容易被带偏：①黑白棋，这个主要是纯MCTS似乎有点困难，可以参考之前学长提供的权重表，以及引入alpha-beta剪枝加上设计好评估函数，能达到不错的效果，测试时记得先手后手还有三个级别都测试一下，比较麻烦的是还要求超出对方的净胜子数，提交时记得勾选报告；②Kmeans，这个最为抽象，所有同学一开始训练出来的模型都是测试后说分数过低，实际上发现测试时有两个case，case1就是有4个错误点，注意构造的特征不要听AI的弄得特别复杂；case2是实际上最终打分的，要求先通过case1，应该是有33个错误点；这个和后面的A股一样，不难注意到测试失败时会输出报错信息，而报错信息是自己的代码产生的（；③A股，这个也已经有很多学长代码了，实际上就是不断调参，这个极其诡异的点是训练集好像只有有1984条，但是测试集有60,000+条；一个办法是直接返回第十四天的作为预测，效果出乎意料的好；当然走捷径的话也可以；④垃圾分类，这个很简单，不用听他的用他那个模型，直接用resnet-18或resnet-34，预训练好的加个分类头练一下就行，最后弄到99.23分感觉也够了；



**大作业**的话，是3-4个人一组，不需要写报告，只需要最后一节课pre（注意问一下老师，自动化似乎和机器人不同，我们今年默认大家都写报告了，后来问了老师才知道不要写，不知道是不是第一节课说了，但ppt上写的是要写报告）今年是双相障碍检测、路标识别和机器人走迷宫，第二个就是一个图像识别问题，第三个似乎就是用Q-Learning，第一个不记得了，我们最终直接选了路标识别。这个大作业就是纯看工作量（主要是这个东西无需也做不出创新），所以我们加了大量的消融实验：1.模型架构：比较基模选择，对比预训练的CNN、Resnet-18、Resnet-34和MobileNetV2（甚至transformer）2.按该模型架构浮动层数、激活函数类型 3.输入分辨率消融32×32、64×64、96×96  4.数据增强消融：无增强、仅基础增强、基础增强 + 颜色扰动、基础增强 + 几何增强 5.优化器消融：SGD、Momentum、RMSProp、Adam、AdamW 6.lr scheduler消融：常数、平方、余弦、指数 7.训练轮数消融；做一个case study，给出一个失败和一个成功案例；在验证集上做一个混淆矩阵，类似相关性分析，行是真值交通标志种类，列是预测交通标志种类，每一格填样本数；由此进一步得到一个表格，训练集每种交通标志样本数量与预测准确率的关系，是否训练样本越多越准确。反正就是纯堆工作量。可以多画一些比较好看的图表之类的。不过感觉大作业可能也不太重要。



对于**期末**，如果想不挂科就行的同学感觉看个1天就可以，想拿比较高分数的同学感觉至少要3天，因为要背要记的东西实在太多了，推荐是如果ppt和我们这届比没变的话直接拿我的笔记看就完全覆盖了，如果有新ppt的话记得看一下。**选择题**都是单选，20道题，每题2分，今年很多大模型的题目，其中有一道题目（第9题）问Transformer中Query、Key、Value、Score哪个包含输入信息，这很诡异哪个能不包含呢？遇到这种题就是没招。其他题大多还挺常规的，有印象的话基本都能选的出来；最难的其实是**填空题**，尤其是今年很多题目极其抽象，会考类似贝叶斯信念网是有向无环图这种冷门问题，目标求解智能体的四个步骤等毫无意义又冷门的问题，26个空26分很多同学反映没几个能写上来的（但是如果想冲高分的同学，我觉得博闻强志的话其实还是可以做到几乎全写出来的）；**判断题**就5道题，除了第一题考的ADA\*复习时候的确没看到之外，其他的也都是容易判断的；最后是**大题**，大题一道题就8分，的第一大题搜索算法写了一张卷子都写不完，不知道为什么不考个两三步一定要用两种启发式搜索完，两个小时其他题半小时就够了这道题花了一个半小时；一个特点是每年卷子的大题重复率真的很高，比如扫地机器人MDP建模（概率转移为1或不为1都有可能考）、八数码A\*问题（用曼哈顿距离或错位数作为启发式函数）几乎是年年必考，小测题也一定会有1-2道作为期末题，比如今年出现的enjoysport种类数分析；alpha-beta剪枝应该也是大题常客，但是去年和今年都没考，可能后面反而需要引起一些注意。总而言之这四道大题至少三道基本考前闭眼都能摸出来，另外一道是一个概念题；



期末有今年今年出现的一个新变化是感觉自动化和机器人的人机考试题几乎趋同，所以学弟学妹在复习时两个专业的卷子都可以看看。实际上我觉得很多题目不是自动化和机器人考点的区别，是每一年ppt都在变化的区别。另外，计算器是不需要的，考的都是概念，或者是简单的整数乘法，可以提前问一下老师需不需要带计算器，如果不需要，那么神经网络反向传播算法的计算、ID3算法、数理统计之类的就自然不会考大题了。
