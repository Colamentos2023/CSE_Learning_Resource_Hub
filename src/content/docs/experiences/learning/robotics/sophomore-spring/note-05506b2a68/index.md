## 前言



大二下学期对机器人的同学而言相比大二上会轻松很多（至少单纯在学业上），一是因为课程难度和数量相对上学期有所下降，二是因为部分课程过于抽象平时不需要听。思前想后，发现实际上这学期走了不少弯路，因此整理了这篇经验分享，希望能帮助到之后学习下列课程的同学们。在正式分享之前，有几点我想先说明：







1. 我希望在类似之前经验分享的基础上，也尽量分享我学习的**基础条件、平时学习和期末准备**的实际情况。每位同学都有自己独特的学习节奏和适应方式，我的经历最多只能帮大家粗略了解一门课可能需要投入的程度，权当是多一个参考视角，不足之处还请批评指正；



2. 部分课程若有必要，我会尽量分析我得分高/低时的具体原因。这些个人案例不一定有普适性，但或许能提供一些对照思考的素材；



3. 需要特别说明的是，这些内容主要反映的是我在**应试和得分**上的观察，会比较功利和急躁。如果同学们对课程内容有更深入的钻研需求，授课老师或其他学长学姐的经验可能会更加有用——这方面我还有很多需要学习的地方。







最后想说的是，毕竟学习方法因人而异，每个人都有自己的独到之处，我的这些经验仅供参考。如果这些分享能给你带来一点点启发，或是帮你节省一些摸索的时间，我就非常开心了。祝愿各位在新的一学期都能学习顺利，事事顺心！

吴俊老师讲机器人学确实非常清楚，至少在Chapter 1到Chapter 7，都能够让人理解在讲什么、要做什么，ppt也算不错；但到了Chapter8和9这一控制部分，或许是因为前置的控制功底不足（实际上讲的内容和自控的关系也不是很大），导致大家完全听不懂在讲什么，需要自己一点点捋清楚，最后我也是只会应付做题的程度。考试前几天感觉全专业的人都在互相问第八章第九章怎么学，结果最后只考了一道选择（？），感觉还是挺抽象的。







机器人学I的巨大计算量想必早已闻名遐迩。这门课的主要内容有：坐标变换和线性代数基础，正运动学、逆运动学、微分运动学、静力学、轨迹规划、动力学和运动控制、力控制，主线是清楚的，就是围绕机械臂的运动，从有目标点开始，进行控制器设计和轨迹规划，分别需要运动学和动力学的计算。对于正运动学，需要重点掌握非标D-H系建立方法、非标D-H参量表的写法；对于逆运动学，我认为无需过于深究PIPPER解法等方法，这一部分上课基本是在讲一些数学技巧，实际的套路就是你用正运动学带着θ、d等变量参数算一遍，对着已知最终的运动学方程矩阵T，类似待定系数法看哪个简单解哪个就行，而且考试历来不怎么考；对于微分运动学，重点是理解几何雅可比、分析雅可比、力雅可比之间的联系与区别，以及雅可比矩阵的四种求法（向外迭代法、向内迭代法、运动学方程求导法、向量积法（注意平动关节和旋转关节不同！），作业习题里会有）；轨迹规划相对比较简单，也很好理解，把关节空间的几种方法（尤其是4-3-4规划和带抛物线过渡段和伪中间点的规划）和笛卡尔空间的slerp方法弄清楚还是很容易的；动力学部分，重点就是两种办法，第一是牛顿迭代法，老老实实带公式一点点推别算错就行，没有思维难度；第二是拉格朗日法，学过理论力学（乙）的同学应该还对上学期期末最后一道题有印象，就是那个方法，比较无脑，计算量相对前者也略小一些。书上和课上在讲拉格朗日方法时讲到了惯性矩阵、什么c符号什么的，我认为完全用不到，只需知道了解即可，无需掌握，将讲这个M、C、G动力学方程的目的只有两个：1.你用前面两种方法得到动力学方程，即算出关节力/力矩τ后能够整理成M、C、G这样的矩阵形式；2.给控制部分作基础。总而言之你知道有这么个东西就行，无需会算；而对于控制部分，我觉得需要理清楚大概的思路，重点是控制律的设计和α-β分解（很重要）、自然约束和人工约束的划分，其他内容我认为记忆理解即可，无需深入研究。







实际上我觉得这门课就是计算量比较大，用的各种符号比较多，不熟练的话容易最开始刚到头疼，感觉很难，等写作业熟练度上来以后，会发现难度和上学期的模电、概统比都略低，并没有传言中说的那么难。尤其是对于考试而言，在明确不考证明题的情况下，从功利的角度出发，我认为对于一切证明过程都没有必要了解的太深入——在这个基础上，这门课实际上就是上课讲公式背景+证明公式，实际上你会用公式和套路就可以了。相比于机器人建模与控制，从目前的几年回忆卷来看，机器人学I历来不重视控制部分，大家可以有针对性地把精力放在前面的内容上，最后两章作业题和基本概念搞清楚即可。







对于平时作业，如果是计算题，我建议认真完成，要搞清楚用了什么方法、什么公式，怎么算的，如果有时间最好自己多动笔算一算，即使很花时间，熟练用好casio的矩阵计算功能；如果是证明题，如果能自己证当然更好，但是我觉得直接对着答案理解了也未尝不可，因为期末不考证明看，也不太会用到证明的思维。







对于小测，一般是四次，虽然不提前通知，但迹象比较明显。每年题目都是不变的，可以提前做一下，难度也不是很大。是要写在一张纸上交就可以，开卷，约等于不管。







对于大作业，基本是要求联系课内理论+轨迹规划+控制器设计+仿真吴老师问问题不会太上压力，也没有必要特别卷。实际上我觉得实际在做的时候，这几个部分联系不是很大，可以各自独立去做，而且尤其是控制器设计在课内没有听明白的情况下，甚至都不太清楚要做什么。最后我还是建议直接用matlab做一下即可，用现成的机器人工具箱很快就能做好，可以考虑再实现一个简单的rrt算法，感觉没有必要太卷，内容完整就够了。我们是一开始不太清楚，就给自己上了强度用issac-sim试图做一个打台球机器人，最后浪费了很多时间，预期一降再降，就结合着matlab和issac-sim两个分别做了一下，实际上都没什么联系，我建议后来的同学除非对一个仿真软件非常熟悉，否则不要盲目去卷或者去探索，性价比不是特别高。当然，如果李老师比较鼓励卷这个的话，再有其他考虑。从身边同学的反应来看，李高峰老师值得大家信任~







对于期末，我是每章看了教材和FrankG的笔记，又把作业题看了一遍，没有做书后其他题；历年卷机器人学和机建的都看了一遍，个别的算了一下，这样对于期末我觉得基本是够了。今年有一半卷子是李高峰老师出的，有两点要注意：1.会考查第一章绪论的内容，复习时候别忘记看一眼；2.不同于吴俊老师出题比较规整，李老师比较喜欢出类似今年最后一题平面几何那种比较“巧招”的题目，所以可能平时思维上也要稍作训练？不过可以保证的是，期末的难度肯定比平时作业要简单，计算量也要小很多（虽然仍然很大）。







这门课从今年开始使用的吴老师自己编写的教材，这本教材由于第一次出版，疏漏有一些，比如轨迹规划一章的公式居然有错等等，但总体上我认为可以视为一本很优秀的讲义，结构很清晰，甚至比较适合自学提升使用（建议课还是要听的）。当然，从明年（2026年）开始，机器人学I将由李高峰老师全英文讲授，**因此目前的经验是否还适用**，会不会还出现新的变化，还请同学们谨慎参考、密切关注。







二编，刚刚出分，感觉有点爆的原因估计大概率还是期末比较炸，毕竟最后一题没做出来（）也有可能是平时分给的比较低，因为身边统计学感觉大家普遍不高，不过不重要了，lgf老师听说给分不错，祝福大家

- **个人感受：** 吴俊老师上课还是一如既往地认真负责，助教老师也很负责，有什么问题都会秒回（）；但对于吴老师在这门课的ppt，我可以明确给出反对的评价：色彩过于丰富，排版过于混乱，结构过于不清晰，重点完全不突出，让人不知所云。而且这门课的作业看上去和课程内容留的有点各自独立（？），经常出现课上讲了的典型方法作业却没有对应的习题（也有可能是内容量太大导致的）。此外，吴老师上课一般是不对着麦克风讲课的，因此我建议如果习惯于听老师讲课而不是习惯自学的同学到课听课；而对于像我一样可能更喜欢自学的同学，可以考虑吴俊老师的智云+声音开到最大或其他老师上课（？），我是看ppt强行看下来的，并自己做了笔记。现在FrankG学长也整理了一份很好的笔记，可以作为很好的学习参考。







整体来说，这门课只是内容很多，绝非难度很大；如果内容少一些的话，我想我会给出5分的难度分——因为和机器人学I一样，从功利角度出发，课上讲的是公式的证明，我们会用公式就可以了。每一部分的内容理解起来也没有很大的难度，和其他几门课比不了。因此我的建议是学习这门课，最好的办法是结合笔记/ppt快速看一遍课，重点搞清楚典型方法和例题，认真做作业，平时跟上尽量不要堆到期末，补天内容量太大了根本学不完；







这门课是结合了自动控制原理（甲）和现代控制原理的几乎全部内容，真的几乎没有任何减少（自控部分只少了第二章的物理系统分析。现控部分少的相对多一点），因此课程内容量极大，讲的也很快，因此连吴老师最喜欢的小测的时间也没有了，基本就是一直在讲课，更别提复习时间了，可以说是“拼好课”。但整本书脉络还是比较清晰的，学习的时候重点搞清楚内在联系。Chapter2到Chapter6是自控的内容，第二章需要搞清楚微分方程、传函、方块图、信息流图、状态空间模型（Chapter8还会细讲）五种表示控制系统的方式以及相互之间的转换；从第三章到第六章，学习时格外注意使用的是**开环传函还是闭环传函**：第三章重点是搞清楚响应的计算和分解、稳态的概念、动态的概念，重点分析欠阻尼系统的动态指标；第四章重点是稳定性的Routh判据（两种0情况处理和稳定裕量判断）、稳态误差的两种计算方法（E(s)终值定理和单位负反馈的误差系数法）；第五章是根轨迹的绘制，需要理解根轨迹的意义，重点掌握绘制方法，以及在根轨迹基础上的临界稳定、临界阻尼等情况分析，根轨迹的校正方法（串联环节），还有重点是广义根轨迹、零度根轨迹，而纯滞后作为了解的处理；第六章重点是波特图的绘制方法，由波特图反推传函，以及开环幅相曲线的绘制、闭环Nyquist图的绘制（格外注意对称和m阶系统的虚线圈补充）和判稳方法（幅角原理什么的不用太管，会用Nyquist判据结论即可），最后是相位裕度、幅值裕度的计算求解；Chapter7到Chapter9是现控的内容，其实难度不大，更多的就是带公式，对于第七章，内容量很大，因为相当于把自控部分的稳快准指标都在离散系统讲了一遍，重点掌握Z变换、脉冲传递函数的求解、差分方程、状态方程和脉冲传函三者的转换，连续状态方程离散化和求解，稳定判据（极点在单位圆内或先用双线性变换再用Routh判据）和最后离散系统的数字化校正方法；第八章是空间状态方程的延伸，包括能控性、能观性的判据（重点是秩判据）、线性变换、结构分解、状态反馈和全维状态观测器设计，都很套路化；第九章重点是描述函数法的稳定性和自激振荡分析，以及李雅普诺夫的稳定概念，间接法、直接法稳定判据、李雅普诺夫方程的使用等等。从内容量和难度综合分析的话，我认为7>6>5>3≈2≈8>4>9







对于平时作业，作业量略大，我们这次助教老师比较善良，批的很松，基本都是给了满，但我还是建议认真做，自己多动笔算算没什么坏处，有助于搞清楚基本概念；







对于实验，这门课的实验课无需到课，只需完成三次matlab实验。期末不会考查，也是完全没有任何指导的，助教老师也是管的不太严，因此可以考虑AI帮帮忙这样子？







对于期末，我建议提前一段时间把基础知识进行梳理，把作业再看看，并可以考虑做一做自控学习辅导那本教材的每章的例题，难度会比考试和作业都大一些，但很有代表性，难而不偏不怪，很适合作为进一步的练习。据吴老师说期末难度是在平时作业之上，在自控甲之下，实际考起来确实也是这个感觉。现控的部分由于讲的比较晚，切不可忽视练习。我也整理了一份A4或许能帮到大家。







出分后问了一下，换算一下期末应该是占比60%，所以失分影响会比较大，看起来主要是有两道大题计算有失误，导致翻车，也警示大家平时一定要注意细节，一些基本概念也要搞清楚，计算也要认真啊~感觉试卷难度本身其实是不大的，做的时候也做的很快，检查的时候居然还没检查出来，还是要练好基本功啊

毛概首先就是靠背，也是背的越多越好，大题思政都是这样，不过我总感觉闭卷的思政课如果你能背出来东西的话更容易拿高分（而且自己语言很难组织；第一类大题是简答题，这种题问的很直接，比如直接问“新民主主义革命的性质、前途、动力”这种比较朴素的点，这个就需要我们去背了；第二类是辨析题，感觉可能错的居多？有几年考的比较难；如果你确定一定是对的或错的（看到过原话），大胆写上判断再写理由；如果不敢确定，就不要说对或错，理由也写的糊里糊涂点，不要说得太明白，把你知道的相关的正确的话写上，也会给分；最后一类是材料题，首先判断材料题指向的是哪一章节哪一个点，知道最好，就先写这个点然后往外扩张相关点；如果不知道就想到什么写什么，写多了又不扣分。







主要是选择题，毛概难就难在选择题，因为还有不定项。这部分一定会考的有两点必须注意：1.教员的各个著作，什么时期写的，主要内容/目的，都集中在那几个小节我记得，考之前一定要看一下熟悉一下，应该也有人整理过；2.历次重大会议及其目标，这里的重大不是只有一大、遵义会议、十一届三中全会这个级别的，像六届六中、六届七中、十四大这类也会考到，需要记住的是会议和会议内容的映射表，因为会考不止一道；3.个别有时候还会出比较抽象的，问一个冷门的例如“十个xx中第一个是什么”，我的建议是有时间的同学翻一下教材里那种是框图的figure，把第一个是什么有个印象，肯定不能问最后一个最不重要的是什么吧x







最后，一定一定要看回忆卷！因为回忆卷里选择题会出原题，我感觉他题库不是很大的，所以现有的几套回忆卷有时间一定要看！
