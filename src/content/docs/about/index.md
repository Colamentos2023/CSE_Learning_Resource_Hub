# 关于项目

Control Nexus（智控学园）是面向控制科学与工程相关课程的静态课程资源库与学业导航平台。

当前版本聚焦三件事：

- 将课程资料按课程名和统一资料类型整理。
- 将学习经验、六级、面试、升学等非课程资料放入更清晰的导航结构。
- 以 Astro Starlight 构建可搜索、可响应式访问、支持暗色模式的 GitHub Pages 网站。

## 版权与使用

本网站仅用于学习交流与资料整理。资料版权归原作者或相应权利人所有；如有问题，请联系维护者处理。

## 维护原则

- 不编造作者、年份、课程评价、适用对象等事实。
- 无法判断的信息统一标注“待补充”。
- 课程资源按课程并列组织，同类资料使用统一模块名称。
- 上传前必须检查隐私与授权风险。
